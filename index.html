Tic Tac Toe

Welcome to Tic Tac Toe! This game takes place on a three-by-three grid between two players. Players place their marker (represented by either an X or O) in one of the nine available spaces on the grid. A player is considered a winner when they successfully place three of their markers in a horizontal, vertical, or diagonal row. The game is considered a draw if all the available spaces have been filled but no declared winner.

This game is set such that the first player uses the "X" marker and the second player uses the "O" marker. Click the "Start Game" button to begin playing.
